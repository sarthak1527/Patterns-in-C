# C Pattern Library

A collection of pattern programs written in C for learners

This repository focuses on improving:
- Nested loop concepts
- Logical thinking
- Pattern-solving skills
- Practical exam preparation

## 📂 Categories Included
- ⭐ Star Patterns
- 🔢 Number Patterns
- 🔤 Alphabet Patterns
- 🔲 Hollow Patterns
- 🔺 Advanced Patterns (Diamond, Butterfly, Spiral, etc.)
